Модальное окно

Функция модального окна, отвлекать внимание пользователей на краткое, важное сообщение. Модальное окно обычно перекрывает основное содержимое страницы, затеняя его и оставляя доступным только содержимое самого модального окна. Остальное содержимое страницы должно оставаться инертным и пользователи не должны взаимодействовать с ним, пока модальное окно открыто.

Техника создания модального окна

Шаг 1: Задать разметку

Одним из популярных способов создания модального окна является центрирование элемента внутри другого элемента растянутого на всю ширину экрана.

Шаг 2: Добавить стили

.modal {
  position: fixed;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  background: rgba(0, 0, 0, 0.4);
  display: none;
}

.modal--open {
  display: block;
}

.modal__content {
  position: absolute;
  background: white;
  padding: 1em;
  left: 50%;
  top: 50%;
  transform: translate(-50%, -50%);
}

Шаг 3: Открытие модального окна

Модальное окно открывается при добавлении класса modal--open к основному элементу.

Шаг 4: Добавить focus-trap

Чтобы закольцевать фокус внутри модального окна, используется техника focus-trap, которая также включает в себя добавление общепринятых комбинаций клавиш и паттернов удобных для управления модальным окном.

Например для закрытия модального окна можно нажать клавишу ESC или кликнуть вне содержимого модального окна.

const button = document.getElementById('button');
const modal = document.getElementById('modal');
const content = document.getElementById('modal-content');

let modalFocusTrap;

button.addEventListener('click', () => {
  modal.classList.toggle('modal--open');

  modalFocusTrap = focusTrap(content, {
    clickOutsideDeactivates: true,
    onDeactivate: () => {
      modal.classList.toggle('modal--open');
    }
  });

  modalFocusTrap.activate();
});
